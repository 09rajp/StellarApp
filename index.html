THE UNIVERSE

The universe is a magical place




This location based app provides information on current astronomical events such as star alignments, constellations, meteor showers, moon phases, sunrise/sunset times, etc. With implemented geographical guides to planetariums, science centers, and museums; we also use APIs to access user location to provide useful, up-to-date information.

You will find multiple tabs for facts, pictures, videos, and future predictions for meteor showers, how far apart, whether they’ll be visible to the naked eye, and so much more!
